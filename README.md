# InterviewGPT

An AI-powered Technical Interview Coach built using RAG (Retrieval-Augmented Generation) and Groq Cloud API (llama-3.1-8b-instant). This application parses resumes, extracts technical skills, offers instant resume optimization advice, and evaluates candidate answers based on specific interview question sets.



## Features

* **Resume Parsing & Analytics:** Upload PDF resumes to automatically extract modern technical skills.
* **AI Recommendations:** Get highly actionable tips from an expert recruiter perspective to improve resume structure.
* **RAG-Based Question Generation:** Upload an interview preparation PDF to securely index contexts and generate target questions.
* **Intelligent Evaluation:** Evaluates candidate answers dynamically on Technical parameters, Communication skills, Strengths, Weaknesses, and provides an Optimized Model Answer.



## Tech Stack

* **Frontend:** Streamlit
* **LLM Provider:** Groq Cloud API (llama-3.1-8b-instant)
* **Frameworks:** LangChain / VectorDB Environment
* **Environment Management:** Python-dotenv
